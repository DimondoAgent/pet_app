# Pet App 🐶

Приложение для взаимодействия пользователей с питомцами (лайки, чат, профиль и т.д.)

## 🚀 Функционал
- Регистрация / логин (Firebase)
- Профили питомцев
- Лайки / дизлайки
- Чат
- Карта

## 🛠️ Технологии
- Python
- Kivy
- Firebase (Auth + Realtime DB)

## ⚙️ Запуск проекта

```bash
git clone https://github.com/DimondoAgent/pet_app.git
cd pet_app
pip install -r requirements.txt
